# ⚖️ LegalConnect — Advocate & Client Platform

A full-stack web platform that connects clients with professional advocates (lawyers), enabling seamless case management, consultations, and communication.

---

##  Overview

LegalConnect provides a digital ecosystem where:

* **Clients** can search for advocates, send requests, and manage legal cases
* **Advocates** can organize clients, handle case files, and communicate efficiently

The platform is designed to simplify access to legal services while improving workflow and organization for legal professionals.

---

##  Tech Stack

### Frontend

* Next.js (React + TypeScript)
* Zustand (state management)
* CSS (custom styling)

### Backend

* Node.js + Express.js
* Supabase (PostgreSQL, Auth, Storage)

### Database

* PostgreSQL (via Supabase)

---

##  Features

###  Client Features

* Search and browse advocates
* Send legal requests
* Book consultations
* Track case progress
* Chat with advocates
* Leave reviews

###  Advocate Features

* Manage incoming client requests
* Accept or decline cases
* Handle consultations
* Organize case files
* Communicate with clients
* View ratings and feedback

###  Core Functionalities

* Authentication (Supabase Auth)
* Real-time messaging (optional upgrade)
* Role-based access (client / advocate)
* Case lifecycle management
* File/document handling

---

##  Authentication

Handled via Supabase:

* Email/password signup & login
* JWT-based session management
* Token used for backend API authorization
